# Physical-Pinball 物理弹球示例 Cocos Creator 2.0.1 + Javascript 版

物理弹球示例：[点击试玩游戏完整demo](https://www.super-cell.club/physical-pinball/)

### 功能点

#### 游戏场景预加载
#### 开局指引动画
#### 辅助射线瞄准
#### 随机生成障碍、随机+递增障碍值、随机形状及颜色
#### 计分、记球、加球
#### 游戏快要over时的抖动提示
#### 障碍物被碰撞时抖动效果
#### 游戏结束判定、重开等...
